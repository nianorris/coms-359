About the Class

About This Class

COMS 359 is an introductory web development course. You will learn how HTML, CSS, and JavaScript work together to create websites that anyone in the world can visit.

By the end of the course you will have built and published your own site — just like this one.

What We Cover

🏗️

HTML

Structure your content with headings, paragraphs, links, images, and more.

🎨

CSS

Style and layout your pages using colors, fonts, flexbox, and grid.

⚡

JavaScript

Add interactivity — respond to clicks, update the page, and handle forms.

🚀

GitHub Pages

Deploy your site live on the web for free using Git and GitHub Pages.
